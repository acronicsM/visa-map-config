# startproject

Запусти проект VisaMap: Docker-контейнеры, бэкенд FastAPI и фронтенд Next.js. Перед запуском каждого компонента проверяй, не запущен ли он уже.

## Структура проекта

- Бэкенд: `visa-map2/` относительно корня воркспейса (FastAPI, порт 8000)
- Фронтенд: `visa-map2-frontend/` относительно корня воркспейса (Next.js, порт 3000)
- Docker: PostgreSQL (порт 5442) + Redis (порт 6379)

Корень воркспейса — это директория, в которой открыт проект. Все команды `cd` выполняй относительно него.

## Шаг 1 — Docker-контейнеры (PostgreSQL + Redis)

Выполни команду для проверки запущенных контейнеров:
```
docker ps --filter "name=visamap_postgres" --filter "name=visamap_redis" --format "{{.Names}}"
```

- Если оба контейнера (`visamap_postgres` и `visamap_redis`) уже присутствуют в выводе — сообщи "✓ Docker-контейнеры уже запущены" и переходи к шагу 2.
- Если хотя бы один отсутствует — выполни:
  ```
  cd visa-map2 && docker-compose up -d
  ```
  Затем подожди 10 секунд и снова проверь статус контейнеров через `docker ps`. Убедись что оба контейнера запущены.

## Шаг 2 — Бэкенд FastAPI

Проверь, работает ли бэкенд:
```
curl -s -o /dev/null -w "%{http_code}" http://localhost:8000/health
```

- Если ответ `200` — сообщи "✓ Бэкенд уже запущен на http://localhost:8000" и переходи к шагу 3.
- Если нет ответа или ошибка — запусти бэкенд в новом терминале (в фоне):
  ```
  cd visa-map2
  uvicorn app.main:app --reload
  ```
  Подожди 5 секунд, затем повтори проверку `curl`. Если ответ `200` — сообщи "✓ Бэкенд запущен". Если нет — сообщи об ошибке и покажи вывод терминала.

## Шаг 3 — Фронтенд Next.js

Проверь, работает ли фронтенд:
```
curl -s -o /dev/null -w "%{http_code}" http://localhost:3000
```

- Если ответ `200` — сообщи "✓ Фронтенд уже запущен на http://localhost:3000" и переходи к итогу.
- Если нет ответа или ошибка — запусти фронтенд в новом терминале (в фоне):
  ```
  cd visa-map2-frontend
  npm run dev
  ```
  Подожди 10 секунд (Next.js компилируется дольше), затем повтори проверку `curl`. Если ответ `200` — сообщи "✓ Фронтенд запущен". Если нет — подожди ещё 10 секунд и проверь снова. При ошибке — покажи вывод терминала.
