# Контекст продукта

## Зачем существует этот проект

VisaMap решает реальную проблему путешественников: узнать, нужна ли виза в конкретную страну — это разрозненная информация. Официальные сайты МИД сложны, Passport Index неудобен для визуализации. Человеку нужен быстрый наглядный ответ.

## Проблемы, которые решает

- **Разрозненность данных**: визовые режимы разбросаны по сайтам МИД и агрегаторам
- **Устаревание данных**: визовые политики меняются, нужен механизм мониторинга изменений
- **Отсутствие планирования**: нет инструмента выбора страны по критериям и построения маршрута
- **Нет визуализации**: таблицы неудобны — карта даёт мгновенное понимание

## Как продукт должен работать

1. Пользователь выбирает паспорт (страну гражданства) из дропдауна
2. Карта мира окрашивается по категориям визового режима
3. Клик на страну — карточка с деталями (категория, срок пребывания, условия)
4. Уровень достоверности данных виден сразу (МИД/модератор/автоматически)
5. В будущем: фильтрация стран по критериям, планирование маршрута, точки интереса

## Целевая аудитория

- Путешественники, планирующие поездку
- Экспаты и цифровые кочевники
- Люди, рассматривающие эмиграцию

## UX-цели

- **Мгновенный ответ**: первый экран показывает карту без регистрации
- **Наглядность**: цветовое кодирование понятно без легенды
- **Доверие**: уровень достоверности данных прозрачен для пользователя
- **Скорость**: GeoJSON кешируется 24h, visa-map 1h — нет задержек при переключении паспорта
