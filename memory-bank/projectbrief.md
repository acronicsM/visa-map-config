# VisaMap — Интерактивная карта визовых режимов

## Суть проекта

Веб-сервис показывающий визовые режимы между странами на интерактивной карте и позволяет установить дополнительные фильтры.
Пользователь выбирает паспорт (страну гражданства) и видит все страны мира окрашенными по категории визового режима. 
Так же сервис позволяет подобрать страну для путешествия по установленным критериям пользователя, спланировать путеществие, делится планами, бронировать отели, самолеты и поезда, показывать точки интереса

## Цели

- Дать пользователям быстрый наглядный ответ: «куда я могу поехать с паспортом страны X?»
- Показывать детали визового режима (максимальный срок пребывания, уровень достоверности данных)
- Поддерживать актуальность данных через RSS-мониторинг новостей и admin API
- Выбирать направление путешествия по критерия
- Создавать маршруты путешествия и календарь путешествия
- Видеть точки интереса
